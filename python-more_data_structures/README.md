Python - More Data Structures: Set, Dictionary
This project focuses on enhancing my understanding of Python programming and its data structures. I will work with sets, dictionaries, and various functions to manipulate them. By completing these tasks, you will strengthen your ability to work with different data structures effectively.

Tasks

0. Squared Simple
Write a function that computes the square value of all integers in a matrix.
 
1. Present in Both
Write a function that returns a set of common elements in two sets.

2. Update Dictionary
Write a function that replaces or adds a key/value pair in a dictionary.

3. Best Score
Write a function that returns the key with the highest integer value in a dictionary.

4. Multiply by Using Map
Write a function that returns a list with all values multiplied by a number using the map function.
